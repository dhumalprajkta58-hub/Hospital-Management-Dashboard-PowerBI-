# Hospital-Management-Dashboard-PowerBI-
Developed an interactive Power BI dashboard to analyze hospital operations, including patient admissions, doctor performance, and treatment outcomes. Utilized data modeling, DAX, and visualization techniques to support efficient healthcare decision-making.
This Power BI project focuses on analyzing hospital operations and patient data to improve decision-making and efficiency. The dashboard provides meaningful insights related to patient admissions, treatments, staff availability, room occupancy, and hospital revenue.
